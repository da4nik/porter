Porter: docker container manager via consul config
=================================================

## Default values:
* **docker sock**: `/var/run/docker.sock`
* **consul api**: `http://localhost:8500`


## Usage help:

`porter --help`

or 

`porter command --help`

## Commands:

* cleanup
* run
* addresses
* config
* set-config
* build
* push
* pull
* listen
